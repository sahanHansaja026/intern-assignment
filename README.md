# Flutter E-Commerce App

A simple and modern Flutter E-Commerce application developed as part of an internship assessment. The application uses local JSON data and demonstrates product browsing, cart management, and a clean user interface without requiring a backend.

---

## Features

### Splash Screen
- Custom splash screen
- Automatically navigates to the login screen

### Login Screen
- Mock login screen
- No authentication required

### Home Screen
- Displays products in a responsive grid
- Product image
- Product name
- Price
- Rating
- Search products
- Category filtering

### Product Details
- Product image
- Product name
- Price
- Description
- Rating
- Add to Cart button

### Cart
- View added products
- Increase quantity
- Decrease quantity
- Remove products
- Automatically calculate total price

### Orders Screen
- Displays mock order history using local JSON data

## Getting Started

### Prerequisites

- Flutter SDK (3.x or later)
- Dart SDK
- Android Studio or VS Code
- Android Emulator or Physical Device

---

## Installation

### 1. Clone the repository

```bash
git https://github.com/sahanHansaja026/intern-assignment
```

### 2. Navigate into the project

```bash
cd flutter_application_1
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the application

```bash
flutter run
```

---

## Build APK

Debug APK

```bash
flutter build apk
```

Release APK

```bash
flutter build apk --release
```

Generated APK location:

```
build/app/outputs/flutter-apk/
```
